# プログラミング

## [YS1](https://yumayx.github.io/YS1/)

コンピューターを操作する上での基本的なAPI、コンセプト

プログラミング言語Rubyを使用して、テキスト操作などの基本的な情報処理を自動化する。デジタル化の第一歩

```ruby
irb(main):001> require 'ys1'
=> true
irb(main):002> 
```

- 言語: **Ruby**
- 成果物: **RubyGem**
- カテゴリ: **効率化**
- URL: <https://yumayx.github.io/YS1/>
